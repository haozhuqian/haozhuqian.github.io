中间件

给 express 创建的实例传入一个回调函数，这个回调函数就是中间件（middleware）

中间件会被传递三个参数

  • 请求对象（request）
  • 响应对象（response）
  • next 函数（在express中定义的用于执行下一个中间件的函数）

中间件中可以执行任何代码，甚至修改 request 和 response 对象，还可以结束响应周期（也可以不结束如果没有调用 next 将控制权交给下一个中间件，这个请求会被挂起），或者执行下一个中间件。

注册

主要有两种方式

  • app/router.use () ;
  • app/router.methods () ;这里的 methods 指 get() 、post() 等方法

如果匹配到的中间件没有执行next(),则执行完本中间件后不执行其他中间件，否则执行下一个匹配的中间件。
